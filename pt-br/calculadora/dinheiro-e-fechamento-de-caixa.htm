Quando usar esta calculadora?

Essa calculadora é muito prática para contar dinheiro, só se preocupe com o número de notas e moedas e deixe os cálculos para nós.

Seja para o fechamento de caixa, uma sangria ou só para contar dinheiro, acelere o processo e minimize os erros, apenas preencha o número correto de moedas e notas que o cálculo é feito imediatamente, e se desejar clique em "Salvar no histórico" caso deseje ver este cálculo mais tarde.

Onde os meus dados ficam salvos?

Somente no seu navegador de internet, não salvamos nenhum dos seus dados em nossos navegadores, então se não quiser perder o histórico cuidado ao limpar os dados do seu navegador.
